O que CSS? - Tecnologia que significa "Cascading style sheet" onde podemos centralizar as configurações de cor, formatação e estilo das páginas. O que facilita a manutenção do código. Resposta Professor: CSS é uma tecnologia utilizada para configurar a aparência e o layout de uma página HTML. Podemos utilizar o CSS de 3 maneiras: Configurando os estilos nas próprias tags HTML, utilizando o atributo style; Configurando os estilos na tag.
