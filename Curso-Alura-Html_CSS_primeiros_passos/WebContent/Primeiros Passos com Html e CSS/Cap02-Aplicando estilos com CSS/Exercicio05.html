overline , line-through Resposta do professor: É possível utilizar quatro valores na propriedade text-decoration: underline, para adicionar uma linha abaixo do texto; overline, para adicionar uma linha acima do texto; line-through, para adicionar uma linha ao centro do texto; e blink, para fazer com que o texto fique piscando. Obs: a opção blink não é suportada por todos os navegadores.
